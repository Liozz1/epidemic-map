#### **简介**：

基于vue的移动端新冠疫情信息和数据展示网站，有疫情实时数据，疫情最新消息, 疫情地图等功能

#### 安装教程：

1.  使用git

```
下载git工具，在对应项目文件夹下右键然后 Git Bash Here
```

2.  克隆项目

```
git clone https://gitee.com/q2513102/vue_shop
```

3.  安装依赖

```
cnpm install
```
